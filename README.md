# Hospital-Management-System-for-minor-project
Hospital Management System This is a Hospital Management System developed in C# that helps manage hospital operations efficiently.The system allows handling of patient records, doctor information, appointments, billing, and staff management.It is good for your college projects 


How to Run the Project in VS Code

1. Install Required Software

Install .NET SDK (latest version)

Install Visual Studio Code

Install C# Extension in VS Code (by Microsoft)

2. Clone the Repository

Open a terminal and run:

git clone <repository-url>


Then open the project folder in VS Code:

cd project-folder-name
code .

3. Restore Dependencies

In the VS Code terminal, run:

dotnet restore

4. Build the Project

Compile the project using:

dotnet build

5. Run the Application

Start the application with:

dotnet run

6. (Optional) Configure Database

Update the database connection string in appsettings.json (if applicable)

Ensure the database server is running

Run migrations if required

7. Access the Application

For Console App: Output will appear in the terminal

For Web App (ASP.NET): Open the provided localhost URL in your browser

For Desktop App (WinForms/WPF): The application window will launch automatically
